## 一、核心工艺总览

| 工艺 | 全称 | 核心作用 | 关键制程 |
|------|------|----------|----------|
| **RDL** | Redistribution Layer（重布线层） | 在晶圆表面重新布置I/O端口，缩小芯片面积 | 沉积、光刻、刻蚀、电镀 |
| **Bumping** | 凸块制作 | 取代Wire Bonding，实现芯片间电气互连 | 焊料沉积、回流、UBM镀层 |
| **TSV** | Through-Silicon Via（硅通孔） | 垂直互连，实现3D堆叠 | 深硅刻蚀、绝缘层沉积、通孔填充 |
| **Fan-out** | 扇出型封装 | 将芯片封装在晶圆/面板上，实现多芯片集成 | 重组晶圆、塑封、重新布线 |
| **CMP** | 化学机械平坦化 | 抛光晶圆表面，实现平坦化 | 抛光垫、抛光液 |

## 九、供应商管理框架建议

### 9.1 关键供应商分类（战略矩阵）

```
                    高供应风险
                        │
         ┌──────────────┼──────────────┐
         │   战略型     │   瓶颈型     │
         │ (Strategic) │  (Critical)  │
 高价值 ─┼──────────────┼──────────────┤
         │   杠杆型     │   日常型     │
         │  (Leverage) │  (Routine)   │
         └──────────────┼──────────────┘
                        │
                   低供应风险
```

- **战略型**（高价值+高风险）：TSV设备（RDL设备、Bumping核心设备）→ 建立战略合作
- **瓶颈型**（低价值+高风险）：感光PI、特种焊球 → 开发多个备选
- **杠杆型**（高价值+低风险）：通用塑封材料 → 谈判降本
- **日常型**（低价值+低风险）：一般化学品 → 优化流程

### 9.2 供应商开发周期建议

| 阶段 | 时间 | 主要工作 |
|------|------|----------|
| 需求分析 | 入职1个月内 | 梳理芯成汉奇BOM，识别关键物料 |
| 供应商识别 | 2-3个月 | 建立候选供应商名单 |
| 初步接触 | 3-4个月 | 发送RFQ，获取初步报价 |
| 技术验证 | 4-6个月 | 样品测试，工艺匹配验证 |
| 商务谈判 | 6-8个月 | 价格、交期、MOQ谈判 |
| 正式导入 | 8-10个月 | 签订合同，PPAP |
| 量产监控 | 持续 | 绩效评估，年度复审 |
