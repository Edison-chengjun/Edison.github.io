# 二、3D DRAM技术演进

## 2.1 三层技术架构

**第一层：单元垂直化**
- 突破平面DRAM的面积极限
- 存储单元垂直排列
- 是3D DRAM的基础

**第二层：4F² 结构**
- 三星VCT（Vertical Channel Transistor）vs 海力士VG（Vertical Gate）
- 缩小DRAM cell面积，提升密度
- 4F² = 4倍特征尺寸的单元面积

**第三层：混合键合（Hybrid Bonding）**
- 铜-铜直接键合（Cu-Cu bonding）
- 互联间距 <10μm
- W2W（Wafer-to-Wafer）或D2W（Die-to-Wafer）
- 是HBM4/HBM5的核心封装技术

## 2.2 TSV——3D DRAM的血管

**TSV（Through-Silicon Via，硅通孔）**
- 贯穿硅片的垂直导电通道
- 是连接堆叠 DRAM 层的"血管"
- 每个HBM堆叠包含数千个TSV
- TSV工艺：深反应离子刻蚀（DRIE）→ 绝缘层沉积 → 铜填充

**如果TSV无法制造：**
→ 产线停产
→ 2.5D封装无法进行
→ 3D IC制造整体受阻

## 2.3 内存墙——AI时代的核心矛盾（白皮书新增）

**核心问题：**
- 大模型参数100B → 1T → 10T演进
- GPU计算力增长速度 >> DRAM带宽增长速度
- 系统能耗越来越多被存储访问占据

**HBM带宽演进：**
| 时期 | 带宽 |
|------|------|
| 2016-2020 | 300-900 GB/s |
| 2021-2024 | 1-2 TB/s |
| **2025-2030** | **4-10 TB/s** |

**结论：2.5D达到带宽极限，3D堆叠是必然趋势**

## 2.4 三大技术路线对比

| 路线 | 代表厂商 | 现状 | 互连距离 | 带宽密度 |
|------|---------|------|---------|---------|
| 2.5D中介层 | NVIDIA/AMD/华为 | 量产成熟 | 偏长 | 中 |
| 3DIC Hybrid Bonding | Intel最激进 | 量产初期 | 极短 | 高3-10倍 |
| 存算一体 | 未来方向 | 预研 | 零距离 | 极限 |

## 2.5 3D核心挑战（白皮书新增）

| 挑战 | 说明 | 解决方案 |
|------|------|---------|
| **良率** | GPU×HBM×Bonding良率叠乘 | 新材料+工艺优化 |
| **功耗密度** | >1 W/mm² | 散热突破 |
| **热阻** | 热流通道受限 | 背面供电(BSPDN) |

## 2.6 中国弯道超车机遇

- 3D DRAM更依赖**蚀刻/沉积/键合**设备，而非EUV光刻机
- 长鑫采用横向堆叠简化工艺
- 大陆光刻受限 → 反而倒逼3D DRAM技术突破
- **GPU厂商（海光/摩尔线程/壁仞）可能绕过2.5D直接进入3D**

# 三、先进封装工艺

## 3.1 封装技术路线图

```
传统封装 → 2.5D封装（中介层）→ 3D IC堆叠 → 混合键合
```

## 3.2 2.5D封装
- **核心：中介层（Interposer）**——一块插在芯片和基板之间的硅/玻璃层
- 芯片通过微凸块（μbump）连接到中介层，中介层再通过TSV连接到基板
- 典型应用：HBM + GPU/CPU通过Interposer封装
- **盛合晶微（SJ Semi）**：国内2.5D封装龙头，85%国产份额

## 3.3 3D IC堆叠
- DRAM芯片层层堆叠
- 层间通过TSV + 混合键合连接
- 核心优势：短互联距离 → 超高带宽、低功耗
- 挑战：散热、良率控制、测试复杂性

## 3.4 混合键合（Hybrid Bonding）
- 铜-铜直接键合（无焊料）
- 键合间距 <10μm，密度远超传统微凸块
- W2W（Wafer-to-Wafer）：两片晶圆直接键合，效率高但要求die size一致
- D2W（Die-to-Wafer）：灵活但工艺更复杂

## 3.5 封装产业链

| 环节 | 龙头厂商 | 国内受益标的 |
|------|---------|------------|
| 2.5D封装 | ASMPT, K&S, TSMC | 盛合晶微、通富微电、长电科技 |
| **混合键合/Bonder** | ASMPT/K&S | **⭐华海清科（中国唯一HB路线图）** |
| 封装测试 | 爱德万、泰瑞达 | 长川科技、华峰测控 |

## 3.6 封装材料

| 材料 | 用途 | 国内标的 |
|------|------|---------|
| **Underfill（底部填充胶）** | 3D堆叠中保护micro-bump | 赛伍技术、鼎龙股份 |
| 清洗材料 | 封装前晶圆清洗 | 安集科技 |
| 粘结材料 | 芯片与基板粘结 | 赛伍技术 |

# 六、投资逻辑链

## 6.1 自上而下推导

```
🤖 AI算力需求爆发
    ↓
💾 HBM产能扩张（海力士/三星/美光）
    ↓
🔧 先进封装需求↑（2.5D封装、混合键合）
    ↓
🏭 中国晶圆厂扩产（中芯国际、长鑫、长江存储）
    ↓
⚙️ 国产设备/材料替代加速
```

## 6.2 三大投资主线

### 主线一：封装先行 🔮⭐
**核心逻辑：** HBM放量 → 先进封装需求爆发 → 封装厂先受益
- **盛合晶微（SJ Semi）** — 2.5D封装龙头，85%国产份额，IPO募资48亿，J2C厂房投产
- **通富微电** — HBM封装合作方，AMD合作丰富
- **长电科技** — 国内封装最强，全系列2.5D/3D能力
- **⭐华海清科** — **Hybrid Bonding设备，中国唯一具备路线图**（白皮书新增）
- **封装材料：赛伍技术（Underfill）、鼎龙股份（粘结材料）**

### 主线二：设备放量 🔧
**核心逻辑：** 晶圆厂扩产 → 设备采购先行 → 国产替代
- **北方华创** — 薄膜沉积设备龙头
- **中微公司** — 刻蚀设备
- **精测电子** — 存储测试，Q3同比+270%
- **中科飞测** — 前道量检测
- **芯源微/盛美上海** — 涂胶显影

### 主线三：存储复苏 💾
**核心逻辑：** NAND/DRAM周期上行 → 存储厂业绩修复
- **兆易创新** — 定制化存储，利基DRAM
- 端侧AI机遇
- 南亚科定制化存储2026年量产
- *长鑫存储（非上市）、长江存储（非上市）*
