📐 平面向量四心问题

重心 · 内心 · 垂心 · 外心 | 向量建模与二级结论

👤 Peter的高中数学笔记 | 📅 整理于 2026-03-28

⚫ 重心 G (Centroid)

三角形三条中线的交点

GA + GB + GC = 0

G = ((x₁+x₂+x₃)/3, (y₁+y₂+y₃)/3)

📌 二级结论

  • 重心到顶点距离 = 2 × 到对边中点距离
  • S△GBC = S△GCA = S△GAB = S△ABC/3
  • 若 O 为任意点，则 OG = (OA+OB+OC)/3
  • PA + PB + PC = 3PG（P为平面上任意点）
    • 中线交点 向量和为0 坐标平均

    🔶 内心 I (Incenter)

    三角形三条角平分线的交点

    a·IA + b·IB + c·IC = 0

    I = (ax₁+bx₂+cx₃)/(a+b+c), (ay₁+by₂+cy₃)/(a+b+c)

    📌 二级结论

  • 内切圆半径 r = 2S/(a+b+c)
  • I 到三边距离相等（内切圆半径）
  • ∠BIC = 90° + ∠A/2
  • 设 BC=a, CA=b, AB=c，则 I 分角线比：BI:IC = c:b
    • 角平分线 内切圆 边长加权

    🟢 垂心 H (Orthocenter)

    三角形三条高的交点

    HA·BC = HB·CA = HC·AB = 0

    向量垂直：HA ⟂ BC, HB ⟂ CA, HC ⟂ AB

    📌 二级结论

  • H 关于各边的对称点在外接圆上
  • 若 O 为外心，则 OH² = 9R² - (a²+b²+c²)
  • 垂心到顶点距离与外接圆半径关系
  • △ABC 为锐角三角形时，H 在内部
  • 向量关系：OH = OA + OB + OC
    • 高线交点 向量垂直 外接圆关联

    🔴 外心 O (Circumcenter)

    三角形三条垂直平分线的交点

    |OA| = |OB| = |OC| = R

    外接圆半径 R = abc / (4S)

    📌 二级结论

  • O 到三顶点距离相等（外接圆半径）
  • 外心在△ABC内部 ⟺ △ABC 为锐角三角形
  • 外心在边上 ⟺ △ABC 为直角三角形（斜边中点）
  • 外心在外部 ⟺ △ABC 为钝角三角形
  • R = a/(2sinA) = b/(2sinB) = c/(2sinC)
    • 垂直平分线 外接圆 等距

    🎯 四心综合问题常用结论

    G、H、O 共线（欧拉线）
    且 OG : GH = 1 : 2
    向量法证明
    GA + GB + GC = 0 → G 为重心
    判定G为重心
    P A + P B + P C = 3P G
    判定I为内心
    ∠BIC = 90° + ∠A/2
    判定H为垂心
    HA·BC = HB·CA = HC·AB = 0
    判定O为外心
    |OA| = |OB| 或 OA = OB

    💡 解题技巧

  • 看到中线 → 优先考虑重心，用 GA + GB + GC = 0
  • 看到角平分线 → 优先考虑内心，注意边长比
  • 看到高/垂直 → 优先考虑垂心，用向量点乘为0
  • 看到垂直平分线/等距 → 优先考虑外心
  • 四心综合题常用 trick：选一个特殊点（如原点）建立向量方程
