## 一、AI正在重塑芯片架构

**核心数据：**
- AI是当前计算领域最大的技术浪潮，影响力远超PC、互联网和移动互联网
- AI模型训练算力增长：GPT-2(2018) → GPT-3(2020) → GPT-4(2023) = 1X → 100X → 10,000X
- 能耗增长：1X → 20X → 250X
- 芯片数量需求：1X → 20X → 100X
- 到2030年，AI可能占美国电力需求的8%
- 单个AI芯片（如NVIDIA H100）功耗高达700W，远超传统CPU

**结论：** AI不仅推动芯片性能升级，也倒逼整个系统从架构层面解决能耗问题。

---

## 二、先进封装成为主战场

**背景：** 传统摩尔定律已逼近物理极限，PPACt（性能、功耗、面积、成本、上市时间）的进一步提升，必须依赖先进封装技术。

**通过3D堆叠和短距离互连，可实现：**
- 功耗降低30~60%
- 延迟减少
- 面积缩小
- 成本降低

**关键认知：** 封装不再是"后道"工艺，而是芯片性能提升的主战场。

---

## 三、传统正面供电（FSPDN）的问题

- 电源和信号线都在正面，互相干扰
- 导致功耗高、面积浪费

---

## 四、背部供电（BSPDN）技术

### 优势：
1. **面积利用率提升30%** — 电源线移到晶圆背面，释放正面布线空间
2. **功耗降低20%**
3. **供电路径短，IR Drop显著降低**
4. **电源与信号物理分离** — 布线工具不再"被电源网格绊倒"，DRC违规减少，设计收敛更快

### 缺点：
1. **工艺复杂度高** — 需新增晶圆减薄（<100nm）、背面TSV加工、双面光刻对准、晶圆键合等步骤
2. **良率风险大**
3. **散热挑战** — 热源仍在正面，但散热结构作用于背面，热路径被拉长
4. **调试与测试工具缺失** — 所有传统测试工具（电子探针、光学检测）都面向正面设计，背部供电网络几乎"不可见"，需重建调试体系
5. **成本前置，投资门槛高** — 需引入晶圆键合、背面研磨、TSV刻蚀、背面金属化等新设备

### 实现方式：
采用硅通孔（μ-TSV / n-TSV）将电源从背面引入晶体管

### 三种技术路径：
1. **BPR** — 埋入式电源轨（Buried Power Rail）
2. **Power Via** — 电源通孔
3. **DBC** — 直接背部接触（Direct Backside Contact）— 性能最佳、工艺最复杂

### 热管理挑战：
背部供电导致热量更难散出

## 关键术语表

| 术语 | 含义 |
|------|------|
| W2W | Wafer-to-Wafer，晶圆对晶圆键合 |
| BSPDN | Backside Power Delivery Network，背部供电网络 |
| FSPDN | Frontside Power Delivery Network，正面供电网络 |
| TSV | Through-Silicon Via，硅通孔 |
| μ-TSV | 微硅通孔 |
| n-TSV | 纳米硅通孔 |
| BPR | Buried Power Rail，埋入式电源轨 |
| DBC | Direct Backside Contact，直接背部接触 |
| PPACt | Performance, Power, Area, Cost, time-to-market |
| DRC | Design Rule Check，设计规则检查 |
| IR Drop | 电压下降（供电网络性能关键指标） |
| CBA | Circuit Bonding Architecture，电路键合架构 |
