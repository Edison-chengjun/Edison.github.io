⚡ 九、3D NAND vs DRAM核心差异

🔩 3D NAND为什么刻蚀占比更高？

  • • 3D NAND需要 100-200层甚至更多 堆叠
  • • 每层都需要：沉积 → 光刻 → 刻蚀
  • • TSV（硅通孔）是HBM封装的必备工艺
  • • 刻蚀深宽比不断提升（>60:1 for 3D NAND）
  • • 长存相对长鑫更依赖国产刻蚀设备

🧠 DRAM为什么光刻占比更高？

  • • DRAM更依赖 光刻分辨率 提升密度
  • • 摩尔定律推进 → 单机价值量持续上升
  • • 先进制程多重曝光 → 工时增加
  • • HBM3/HBM4的TSV孔洞需要高精度光刻
  • • 高端ArF浸没光刻机是关键
