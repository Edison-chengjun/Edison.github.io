## 一、公司概况

### 1.1 基本信息
- **全称：** 盛合晶微半导体有限公司
- **简称：** 盛合晶微、SJ Semi
- **定位：** 中国大陆领先的晶圆级先进封测企业
- **地位：** 2024年跻身全球前十大OSAT，中国大陆2.5D集成市场占据约85%国产份额

### 1.2 历史沿革
- **2014年：** 由中芯国际（SMIC）与长电科技（JCET）合资设立

- **股东结构：** 无锡产发基金、招银系、深圳远致一号等；无实际控制人状态
- **管理团队：** 由资深半导体专家崔东领衔，核心团队平均20年以上经验

## 二、核心业务与技术平台

### 2.1 战略定位："中道封装"定义者
盛合晶微定义并占领了"中道封装"市场：
- 处于前段晶圆制造与后段传统封测之间
- 主要包含Bumping（凸块制造）、CP测试（晶圆中测）及晶圆级封装技术
- **中国大陆首家提供14nm先进制程Bumping服务的企业**

### 2.2 核心技术平台：SmartPoser®

#### SmartPoser-Si（2.5D/TSV）平台
**用于集成逻辑芯粒与HBM，是高性能计算领域的核心护城河**

| 技术参数/特性 | 详细描述 | 核心优势 |
|---|---|---|
| 重布线层(RDL) | 最小线宽/线距可达2um/2um | 实现极高布线密度 |
| 互联层数 | 支持4P4M结构 | 多层级信号路由，增强设计灵活性 |
| 集成规模 | 支持4倍光罩尺寸的大面积集成 | 突破单芯片尺寸限制 |
| 硅片减薄 | 优于行业标准的减薄厚度 | 改善散热，减小封装高度 |

#### SmartPoser-RDL（扇出型封装/Fan-out）平台
- 无偏差拼接技术：多掩模版精准拼接，有机转接板大面积布线
- 低翘曲控制：热处理变形控制领先
- 异构集成：支持4个以上功能芯片高密度集成

#### SmartPoser-POP 与 SmartAiP（3D集成与5G方案）
- **SmartPoser-POP：** 多层芯片三维堆叠，用于高端手机AP、可穿戴设备（智能眼镜、智能手表）
- **SmartAiP（天线封装）：** 集成5G毫米波天线、芯片及电压转换元件
  - 频率覆盖：24GHz至43GHz超宽带
  - 优势：更低功耗、更高增益、更薄廓形

#### 3DIC-HB（混合键合）与亚微米互联技术
- **混合键合(Hybrid Bonding)：** 铜-铜直接键合替代焊球微凸块，互联间距<10um
- **亚微米互联：** 借鉴前段大马士革工艺，三层及以上堆叠
- **微硅通孔(Micro-TSV)：** 开口尺寸<5um的垂直连接

## 八、核心结论

### 8.1 三大核心优势

| 优势 | 体现 |
|------|------|
| **技术统治力** | 国内极少数能在2.5D/3D领域与台积电、安靠竞争的OSAT厂商 |
| **业绩爆发力** | 芯粒集成三年从5%→56%，2025年利润预期9.23亿元 |
| **供应链闭环** | 高比例国产化的先进制程封测线，降低中断风险 |

### 8.2 未来展望（2026-2030）

**技术下探：**
- 从微凸块向混合键合（Hybrid Bonding）跨越
- HBM4时代核心竞争力的底层保障

**市场拓宽：**
- 汽车电子（自驾芯片）
- 高端可穿戴设备
- 优化客户结构，分散集中度风险

## 九、投资逻辑关联

### 9.1 与精测电子、中科飞测的关系
- 盛合晶微是**封装厂**，精测电子是**检测设备商**，中科飞测也是**检测设备商**
- 封装厂扩产 → 带动检测设备需求
- 三者在同一条产业链上

### 9.2 封装产业链梳理

```
IC设计（客户A等）
    ↓
晶圆制造（中芯国际）
    ↓
中道封装（盛合晶微）← 这里！
    ↓
后道封测（长电科技、通富微电）
    ↓
检测设备（精测电子、中科飞测）
```

### 9.3 存储封装关联
- 盛合晶微的SmartPoser-Si平台用于**HBM与逻辑芯粒集成**
- 受益于AI算力芯片需求爆发
- 与通富微电在HBM封装领域存在协同
